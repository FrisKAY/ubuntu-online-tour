ပလႅၼ်ႇတႃႇ

ပလႅၼ်ႇတႃႇၼႆ့ မၼ်းႁူမ်ႈႁွမ်းၵေႃႇသၢင်ႈဝႆ့တႃႇ ၽၢင်ႈႁၢင်ႈ 3d ၊ လွင်ႈတူင့်ၼိုင်၊ လွင်ႈလျွၼ်းတႃး လွင်ႈႁဵတ်းဢွၵ်ႇ ငဝ်းႁၢင်ႈ ၸိူဝ်းပဵၼ်ၵဵမ်းလဵၼ်ႈလႆႈ လႄႈသင် ၸိူဝ်းပဵၼ်တႃႇငဝ်းတူင့်လႄႈသင်ၼႆၶႃႈယဝ့်။ ပလႅၼ်ႇတႃႇၼႆ့ မၼ်းမီး ပိူင်ၽၢင်ႁၢင်ႈႁင်းၸဝ်ႈၵဝ်ႇ ၸိူဝ်းဢၼ်ၵေႃႇသၢင်ႇ ၸွမ်း OpenGL လႄႈ ၸိူဝ်းၸၼ် Design ဝႆ့ ၸွမ်းၼင်ၸႂ်မီးၼၼ့်ၶႃႈယဝ့်။ တွၼ်ႈတႃႇတေ သူင်ႇဢွၵ်ႇ ႁႂ်ႈမၼ်းၵႂႃႇၸႂ့်လႆႈ တီႈၼႂ်း 3D Studio လႄႈ Wavfront ၼၼ့်ၵေႃႈ မၼ်းမီးပႃးဝႆ့ Python Bindings ဢၼ်တႃႇတႅမ်ႈ Script ၼႆၶႃႈယဝ့်။ တွၼ်ႈတႃႇတေႁဵတ်း ႁၢင်ႈတၢႆ လႄႈသင် တွၼ်ႈတႃႇတေႁဵတ်း ႁၢင်ႈတူင့်ၼိုင် တွၼ်ႈတႃႇ ၵၢၼ်ၸႂ့်တိုဝ်းၵူႈလွင်ႈလွင်ႈၼႆ့ ပလႅၼ်ႇတႃႇ မၼ်းၸၢင်ႈႁဵတ်းလႆႈႁင်းၵူၺ်းမၼ်း ဢမ်ႇဢိင်ၼိူဝ်ၽႂ်သေတႃ့ မၼ်းၸၢင်ႈႁဵတ်းလႆႈ ႁင်းၵူၺ်းမၼ်းယူႇ။
